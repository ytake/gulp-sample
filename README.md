## install
required node.js

## npm

```bash
$ npm install -g gulp
$ npm install -g bower
#
$ npm install
```

## composer
```bash
$ composer install
```
## gulp tasks
■built in server
```bash
$ gulp boot
```
■default
bower install, ファイル配置, コンパイル, phpunitなどが含まれています
